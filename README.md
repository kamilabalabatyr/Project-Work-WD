# Project-Work-WD
Project Description

Project Title: Rental Housing Platform (Airbnb-style)

This project is a web application that allows users to browse, publish, and manage rental properties such as apartments, houses, or rooms. 
The platform is inspired by Airbnb and focuses on a simple and minimalistic user interface.

Users can create an account, log in, and explore available properties listed by other users. 
Each property includes important information such as title, description, price, location, and images. 

The application consists of a frontend built with Angular 
and a backend powered by Django REST Framework. 
The frontend communicates with the backend through REST APIs to fetch and manage data. The backend handles authentication, data storage, and business logic.

Main Features
- User registration and login 
- Viewing a list of rental properties
- Viewing detailed information about a property
- Creating, editing, and deleting property listings
- Basic search or filtering
- User dashboard to manage personal listings
  
Technologies
Frontend: Angular
Backend: Django + Django REST Framework
Database: PostgreSQL / SQLite
Authentication: JWT
API communication: REST API

Goal of the Project
- The goal of this project is to practice full-stack web development by building a functional web platform with a modern frontend framework and a RESTful backend API.
